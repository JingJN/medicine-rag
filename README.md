# MSL 问答 RAG Demo

此 demo 模拟医药公司场景：医生通过微信或邮件询问药品相关专业问题，MSL 不再手动翻说明书、PPT 和文献摘要，而是由 RAG 脚本在本地知识库中秒级检索并生成可审阅答复。

> 说明：本项目使用自造的模拟医学资料，仅用于演示 RAG 工作流，不构成真实诊疗或用药建议。

## 覆盖的问题维度

- 药品不良反应：例如阿司匹林出血风险、瑞舒伐他汀肌痛
- 禁忌与慎用：例如消化性溃疡、哮喘、妊娠晚期
- 药物相互作用：例如阿司匹林+华法林、氯吡格雷+奥美拉唑
- 特殊人群：例如肾功能不全、老年、妊娠和哺乳
- 用法用量：例如阿司匹林心血管二级预防剂量
- 用药过量：例如阿司匹林中毒表现和处理原则
- 临床文献摘要：例如阿司匹林二级预防、二甲双胍疗效特点
- 医学事务合规：例如 MSL 答复边界、引用和升级流程

## 快速运行

运行内置演示问题：

```bash
python3 msl_rag_demo.py --demo
```

询问单个问题：

```bash
python3 msl_rag_demo.py --question "氯吡格雷和奥美拉唑联用会不会影响抗血小板效果？"
```

进入交互模式：

```bash
python3 msl_rag_demo.py
```

## Demo 流程讲解

1. 医生提出自然语言问题。
2. RAG 检索器从知识库中召回相关片段。
3. 脚本对片段进行排序，生成结构化答复。
4. 每条答复保留来源引用，方便 MSL 审阅后再发给医生。
